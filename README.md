# Plant Disease Detection System for Sustainable Agriculture 🌱
This project leverages machine learning and deep learning techniques to develop an intelligent system capable of detecting plant diseases from images. The goal is to aid farmers and agricultural professionals in identifying diseases early, ensuring timely intervention for sustainable farming practices.

# Key Features:
Deep Learning Model: A trained model built with TensorFlow/Keras for high-accuracy disease detection.
Image Classification: Identifies various plant diseases using leaf images.
User-Friendly Interface: Includes a Streamlit web app for easy interaction.
Sustainable Agriculture: Promotes better crop health management to reduce losses.

# Technologies Used:
Python: Core programming language.
TensorFlow/Keras: For deep learning model development.
Streamlit: To create an interactive web interface.
OpenCV: For image preprocessing.
NumPy & Pandas: For data manipulation.

# How to Use:
Upload a clear image of the plant leaf.
The system processes the image and predicts the disease category.
Receive actionable insights for disease management.

# Dataset:
The project uses a curated dataset of plant leaf images categorized by disease type for training and evaluation.

# Applications:
Early detection of plant diseases.
Minimizing crop losses and improving yields.
Assisting farmers in adopting precision agriculture practices.
